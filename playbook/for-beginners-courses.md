# For Beginners Courses

Our highest-impact content — open-source curricula on GitHub with millions of views and tens of thousands of stars. If your product fits into one of these (or warrants a new one), this is a big deal.

## Overview

| Attribute | Details |
|-----------|---------|
| **Impact Level** | ⭐ Very High |
| **Lead Time** | 1-3 months |
| **Effort Required** | High |
| **Best For** | Long-term adoption, education, onboarding learners |

## What These Look Like

- 12-24 lessons organised into modules
- Written content + code samples + embedded videos
- Exercises and quizzes
- Codespaces support so learners can start immediately

## Ways to Get Involved

### Add to an existing course
Easiest path. Add supplementary resources, contribute a new lesson, or reference courses as prerequisite learning for your content.

### Create a new course
Bigger commitment. You'll need to identify a gap, plan the curriculum, create samples that work in Codespaces, write lesson content, and add videos.

## Works Well With

- **Reactor Livestreams** — do a series walking through each chapter
- **Blogs** — write companion posts with embedded lesson videos
- **AI Tour Workshops** — use course content as workshop foundations
- **YouTube** — record lesson walkthroughs

## Tips

- **Support Codespaces** — learners want to start coding without setup
- **Pin your dependencies** — version lock everything to prevent breakage
- **Think like a beginner** — don't assume prior knowledge
- **Plan for maintenance** — technologies change, and broken courses lose trust
