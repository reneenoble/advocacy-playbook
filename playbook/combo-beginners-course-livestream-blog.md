# Combination: Beginners Course + Livestream Series + Blogs

One topic becomes a course, a livestream series, and blog posts — maximum reuse from a single content investment.

## Overview

| Attribute | Details |
|-----------|---------|
| **Combined Impact** | High |
| **Total Lead Time** | 5-8 weeks |
| **Total Effort** | Medium-High |
| **Best For** | New technology onboarding, foundational learning paths |

## The Flow

1. **Design course structure** — identify 3-5 key modules
2. **Create initial course content** on Microsoft Learn
3. **Schedule Reactor livestreams** that deep-dive each module topic
4. **Deliver livestreams** — answer community questions in real-time
5. **Write blog posts** — one per livestream, embed the video recordings
6. **Update course** — embed relevant livestream videos back into course modules

## Why This Works

- **Content multiplier** — one topic becomes course + livestream + blog(s)
- **Community engagement** — livestreams allow real-time Q&A
- **Multiple entry points** — learners discover via course, YouTube, or blog
- **Evergreen** — course stays fresh with embedded video demonstrations
