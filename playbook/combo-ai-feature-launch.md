# Combination: AI Feature Launch

A week-long rhythm for launching AI features: demo Monday, blog Tuesday, AMA Friday.

## Overview

| Attribute | Details |
|-----------|---------|
| **Combined Impact** | Medium-High |
| **Total Lead Time** | 2-3 weeks |
| **Total Effort** | Medium |
| **Best For** | AI/ML feature launches, new model releases |

## The Flow

1. **Coordinate with Model Mondays** — contact April Gittens; Nitya Narasimhan hosts
2. **Record your demo** — 10-15 minutes of focused content
3. **Draft blog post** — include code samples, leave space to embed video later
4. **Monday: demo airs** — share links in stream chat
5. **Tuesday: publish blog** — embed the video, add key takeaways
6. **Friday: Discord AMA** — join [Microsoft Foundry Discord](https://discord.gg/nTYy5BXMWG), answer questions for 30 minutes
