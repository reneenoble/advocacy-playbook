# Blogs

Blog posts drive SEO traffic, give written context to video content, and provide permanent, linkable references. Especially powerful when paired with a video.

## Overview

| Attribute | Details |
|-----------|---------|
| **Impact Level** | Medium |
| **Lead Time** | 1-2 weeks |
| **Effort Required** | Medium |
| **Best For** | SEO, deep dives, tutorials, driving traffic |

## Where to Publish

| Platform | Best For |
|----------|----------|
| [Dev Blogs](https://devblogs.microsoft.com/) | Technical deep dives, announcements |
| [Tech Community](https://techcommunity.microsoft.com/) | Articles, Q&A, community discussions |
| [Developer Blog](https://developer.microsoft.com/blog) | Cross-cutting announcements, integrations |
| Product-specific blogs | Check if your product area has one |

## The Blog + Video Pattern

This works really well:

1. Do a Reactor livestream or record a video
2. Write a blog post that embeds the video
3. Add written context and links to resources
4. Blog drives SEO traffic → video provides engagement

## Works Well With

- **Videos** — embed video content in your post
- **For Beginners** — write posts introducing course content
- **Newsletters** — submit blog posts to newsletters
- **Samples** — link to GitHub repos with more context

## Common Post Types

- **Announcement** — what's new, why it matters, how to get started
- **Tutorial** — problem, step-by-step instructions, code samples, embedded video
- **Video companion** — brief context, embedded video, key takeaways, resource links

## Tips

- Use keywords developers actually search for
- Include code samples — developers want to see code
- Link generously to samples, docs, and related content
- Keep posts updated when things change
