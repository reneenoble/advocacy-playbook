# Advocacy Playbook

A handbook for promoting advocacy techniques within Microsoft and GitHub product and tool teams.

This playbook helps advocacy teams effectively promote samples, demos, and tools to developers and technical audiences.

## Table of Contents

- [Getting Started](docs/getting-started.md) – Quick-start guide for new advocates
- [Promotion Channels](docs/channels.md) – Social media, blogs, conferences, and more
- [Advocacy Techniques](docs/techniques.md) – Demos, samples, tutorials, and storytelling
- [Samples & Demos](docs/samples-and-demos.md) – Best practices for building and sharing samples
- [Measuring Success](docs/measuring-success.md) – Metrics and impact measurement

## About This Playbook

This playbook is maintained by advocacy teams at Microsoft and GitHub. It is intended as a living document — contributions and updates are welcome.

See [Getting Started](docs/getting-started.md) to begin.
